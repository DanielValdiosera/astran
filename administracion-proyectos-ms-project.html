Administración de Proyectos con Microsoft Project

Objetivo

Conocer y aprender la funcionalidad y uso general de Microsoft Project Professional & Online como herramienta de colaboración, y fortalecer la cultura de Gestión de Proyectos aplicando las mejores prácticas. MS Project es una solución para planear, priorizar y administrar proyectos e inversiones.

Formato

Este curso está disponible en modalidad presencial:

  • Administración de Proyectos con MS Project Professional | 16 Horas
  • Administración de Proyectos con MS Project Professional & Online | 24 Horas

Dinámica

  • El facilitador expone una breve explicación de cada concepto, se comentan casos reales para sensibilizar al grupo y se realizan dinámicas en las que se aplica lo expuesto.
  • Las prácticas del taller estarán orientadas a desarrollar un proyecto ejemplo acordado previamente con el responsable del grupo para comprender cómo se aplican las mejores prácticas emitidas por el PMI®, utilizando la suite de herramienta de Microsoft Project

Beneficios

Extender el conocimiento

de los procesos emitidos por el PMI®.

Mejorar el aprovechamiento

de las herramientas de Microsoft para la gestión de proyectos y colaboración.

Facilidad para administrar cronogramas

en el reporte de desempeño para determinar el estado del proyecto y contar con información oportuna para implementar acciones de control.

Perfeccionar la planeación

de tiempos y uso de recursos.
